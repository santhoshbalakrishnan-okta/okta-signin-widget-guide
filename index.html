Add logo
  1. Find // config section under <script> tag.
  2. Add the logo key-value pair to var config json.

var config = {
  baseUrl: '{url}',
  ...
  logo: '/path/to/logo.png',
  ...
};
Change background
  1. Add <style> tag under <head> tag if it does not exist already.
  2. Within the <style> tag, add the following 

#okta-sign-in {
  background-color: insert-color-here
}
